Gets a callable native syscall pointer.

As shown below, the returned function pointer is directly callable thanks to a generic cast. However, this generic cast forces to explicitly cast the value returned by the syscall (if any).

NTSTATUS status = (NTSTATUS)MDK_NTSYSCALL(NtDelayExecution)(FALSE, &delay);
